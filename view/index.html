Welcome!

This is the proxy server I made for my Korean dictionary app. Oh, but you being here means that you probably know about it, hehe. Or don't you? Well if you don't - here's link

I spent quite a lot of time working on both this server and the app (I started around 20th October 2016). I didn't think it would be easy and oh boy, it sure wasn't! But hey, it's alive and kickin' right now and that's what matters!

Anyway, I made this app because the original Naver Dict app was...well, not good enough for me. No, scratch that, it just sucks. It is a glorified browser for Naver's website and I don't like glorified browsers packaged as apps. I really don't. Websites on mobile are clunky, laggy and they're overall hard to use. That's why I took it upon myself to make this app in order to deliver a better user experience. And I did it!

And here we are now, you and I, staring at each other and thinking "What's this guy's deal?". Well if you've come so far, then I probably won't feel bad to say this, so here goes nothing!

As you already know, I spent a lot of time building this app. Unfortunately, since it uses data from Naver and I don't know if monetizing that is legally valid or not, I removed all ads. However if you feel like I deserve some compensation for my hard work, feel free to make a donation! (On the payment page it will show up as if you're buying an item - A Developer's Heart. I may not be able to physically give you my heart, but I will never forget that you've supported me. :) )
